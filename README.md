# Elevate Asset Management Website

A modern, professional property management website built with React and Vite.

## Features

- **Home Page**: Hero section with company logo, tagline, and call-to-action buttons
- **Properties Page**: Showcase of 3 rental properties with before/after photo comparisons
- **About Us Page**: Company story, mission, and values
- **Contact Page**: Contact form, phone, and email information
- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern UI**: Clean, professional design using brand colors

## Brand Colors

- Primary: #2f241D (dark brown)
- Secondary: #ffc425 (yellow/gold)
- Accent: #ff6600 (orange)

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:3000`

### Building for Production

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` folder.

To preview the production build:

```bash
npm run preview
```
